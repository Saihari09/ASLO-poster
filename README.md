# ASLO Poster GitHub Pages Site

This repository is intended to publish the ASLO poster resource page from the
`docs/` directory.

## Publish on GitHub Pages

1. Create a GitHub repository named `ASLO-poster`.
2. Push this folder to that repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the branch you use, usually `main`, and set the folder to `/docs`.
6. Save.

The page will be available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/ASLO-poster/
```

Use that final URL for the poster QR code. Once the QR code is printed, keep
the repository name and GitHub username stable; the page contents can still be
edited later.

## Update Links

Edit `docs/index.html` and replace the placeholder links in the
`Project links` section with the final poster PDF, data release, code
repository, or extended figure URLs.
